## Описание

Приложение, которое загружает список товаров и позволяет пользователю удобно взаимодействовать с данными: сортировать, редактировать и добавлять новые позиции.

---

## API

В качестве источника данных использовалось публичное API [DummyJSON Products](https://dummyjson.com/docs/products).
Для авторизации использовалось [DummyJSON Auth](https://dummyjson.com/docs/auth).

```bash
Данные для входа: 
логин: emilys, пароль: emilyspass
логин: michaelw, пароль: michaelwpass
```

## Функциональность

- **Форма входа**: реализована валидация полей. При ошибке API выводится сообщение. Данные для входа запоминаются: 1. при установленном чекбоксе сохраняется токен авторизации, обеспечивая сохранение сессии после закрытия браузера. 2. если не установлен — сессия сбрасывается при закрытии вкладки.
- **Поиск товаров**: debounce при отправке запроса, прогресс-бар при подгрузке данных.
- **Список**: прогресс-бар при подгрузке данных в таблице. Сортировка по цене и рейтингу. Товары с рейтингом ниже 3 подсвечиваются красным цветом.
- **Добавление**: при нажатии на кнопку "Добавить" появляется новая строка с возможностью редактирования основных полей (имеется валидация): наименование, вендор, артикул, цена. При успешном добавлении в нижней части экрана отображается Toast-уведомление, а также в хэдере. Клик по колокольчику открывает список аналогичных уведомлений для быстрого просмотра новых сообщений.
- **Редактирование**: можно редактировать ячейки в таблице, кроме "Оценки" и "Количества". Отредактированный товар заменяет оригинальный. Ширина столбцов меняется. Все изменения, включая сортировку и состояние столбцов, сохраняются в локальном состоянии.
